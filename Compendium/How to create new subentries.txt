### How to create a new subentry ###

1. Create a new model
1.1 Make migrations
2. Create a new view
3. Create a new HTML file copied from another version
4. Add the new subentry tags to the codeflask.min.js file
5. Add the subentry tags to the prism.css and prism.js files
6. Create a new URL for the link 
7. Add the URL to the base.html file 
8. Add the new tags to the dictionary in signals function
